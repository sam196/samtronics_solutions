# Samtronics Solutions Ecommerce Website

This is an ecommerce website for Samtronics Solutions, specializing in LED signages in Kenya.

## Features

- Product catalog with LED signages
- Shopping cart with local storage
- Responsive design
- Contact form for quotes

## How to Launch

Open `index.html` in a web browser.

## Structure

- `index.html`: Main page
- `styles.css`: Stylesheet
- `script.js`: JavaScript for cart functionality
- `images/`: Product images
